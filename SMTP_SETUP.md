# Настройка SMTP для формы записи на приём

## Создание файла `.env.local`

Создайте файл `.env.local` в корне проекта со следующими переменными:

```env
# SMTP Configuration
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
CLINIC_EMAIL=clinic@example.com
```

## Настройка для разных почтовых сервисов

### Gmail

1. **SMTP настройки:**
   - `SMTP_HOST=smtp.gmail.com`
   - `SMTP_PORT=587`

2. **Получение App Password:**
   - Перейдите в настройки Google аккаунта
   - Безопасность → Двухэтапная аутентификация (включите если не включена)
   - Пароли приложений → Создать новый пароль
   - Используйте сгенерированный пароль в `SMTP_PASSWORD`

### Yandex

1. **SMTP настройки:**
   - `SMTP_HOST=smtp.yandex.ru`
   - `SMTP_PORT=465`

2. **Настройка:**
   - Используйте обычный пароль от почты
   - Или создайте пароль приложения в настройках безопасности

### Mail.ru

1. **SMTP настройки:**
   - `SMTP_HOST=smtp.mail.ru`
   - `SMTP_PORT=465`

2. **Настройка:**
   - Включите IMAP/SMTP в настройках почты
   - Используйте обычный пароль от почты

## Проверка работы

После настройки переменных окружения:

1. Перезапустите dev-сервер: `npm run dev`
2. Откройте форму на сайте
3. Заполните и отправьте тестовую заявку
4. Проверьте почту клиники (`CLINIC_EMAIL`)

## Безопасность

⚠️ **Важно:**

- Файл `.env.local` уже добавлен в `.gitignore`
- Никогда не коммитьте файл с реальными паролями
- Используйте пароли приложений, а не основные пароли
